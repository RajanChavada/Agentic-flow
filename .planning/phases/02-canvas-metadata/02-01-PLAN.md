  <behavior>
    ## GraphMetrics interface

    analyzeGraph(nodes: Node[], edges: Edge[]) returns:
    ```typescript
    {
      nodeCount: number;           // total nodes including annotations
      workflowNodeCount: number;   // excludes blankBoxNode, textNode
      maxDepth: number;            // longest BFS path from startNode
      loopCount: number;           // number of SCCs found via DFS
      toolRiskSurface: { read: number; write: number; exec: number; network: number; other: number };
      riskScore: number;           // aggregate point score
      riskLevel: 'Low' | 'Medium' | 'High';
      idealStateReachable: boolean | null;  // null = no ideal state node on canvas
    }
    ```

    ## Test Cases

    ### Node Count (META-01)
    - Empty graph (no nodes) -> nodeCount: 0, workflowNodeCount: 0
    - 3 workflow nodes + 1 blankBoxNode + 1 textNode -> nodeCount: 5, workflowNodeCount: 3
    - Annotation nodes (blankBoxNode, textNode) excluded from workflowNodeCount but included in nodeCount

    ### Max Depth (META-01)
    - Empty graph -> maxDepth: 0
    - Single startNode, no edges -> maxDepth: 0
    - Linear chain: start->agent->finish -> maxDepth: 2
    - Diamond: start->(A,B)->finish -> maxDepth: 2
    - Deep chain: start->n1->n2->n3->n4->n5->finish -> maxDepth: 6
    - No startNode present -> maxDepth: 0
    - Condition node branching: start->condition->(true_path->finish, false_path->agent->finish) -> maxDepth: 3 (longest path)

    ### Cycle Detection (META-01)
    - Linear DAG -> loopCount: 0
    - A->B->A cycle -> loopCount: 1
    - Two independent cycles -> loopCount: 2
    - Disconnected subgraph with cycle (not connected to start) -> still detected, loopCount: 1
    - Self-loop (A->A) -> loopCount: 1

    ### Tool Risk Surface (META-02)
    - No tool nodes -> { read: 0, write: 0, exec: 0, network: 0, other: 0 }
    - Tool with toolCategory "retrieval" -> read: 1
    - Tool with toolCategory "database" -> write: 1
    - Tool with toolCategory "code_execution" -> exec: 1
    - Tool with toolCategory "api" -> network: 1
    - Tool with toolCategory "mcp_server" -> network: 1
    - Tool with no/unknown category -> other: 1
    - Mixed tools: 2 retrieval + 1 code_execution + 1 api -> { read: 2, write: 0, exec: 1, network: 1, other: 0 }

    ### Risk Score (META-03)
    - Empty/minimal canvas -> score: 0, level: "Low"
    - 1 exec tool -> score: 2, level: "Low" (threshold is 0-3)
    - 2 exec tools -> score: 4, level: "Medium" (threshold is 4-7)
    - 2 exec + 2 network tools -> score: 8, level: "High" (threshold is 8+)
    - 1 write tool only -> score: 1, level: "Low"
    - Graph depth > 5 -> +2 points added
    - Any loops (loopCount > 0) -> +2 points added
    - Total workflow nodes > 15 -> +1 point added
    - Combined: 1 exec(2) + depth 6(2) + 1 loop(2) + 16 nodes(1) = score 7, level: "Medium"

    ### Reachability (META-04, ESTM-04)
    - No idealStateNode on canvas -> null
    - No startNode on canvas, idealStateNode present -> false
    - start->agent->idealState (connected path) -> true
    - start->agent, idealState disconnected -> false
    - start->condition->true_path->idealState (reachable via one branch) -> true
    - Complex graph: start connected through multiple hops to idealState -> true
    - Handles edge case: both start and idealState exist but no edges -> false
  </behavior>
